# Advanced Networking Self Quiz

Score 1 point for each correct answer.

## Questions

1. What is the core runtime rule for this topic?
2. Name one common trap.
3. Name one prevention pattern.
4. Which input edge case matters most?
5. What should you do before refactoring?
6. What should you do after each change?

## Suggested Answers

1. The lesson-specific rule that controls runtime behavior.
2. Any trap that causes misleading output or hidden bugs.
3. Explain-first, const-first, or small-step refactoring.
4. Empty, invalid, or boundary input.
5. Clarify behavior and expected output.
6. Run checks and validate assumptions.
