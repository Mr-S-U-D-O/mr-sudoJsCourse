# 01-var

## Learning First
This lesson is teaching-first. Start with `lessons/` and use `challenges/` only for lightweight practice.

## Folder Guide
- `lessons/`: explanation, visualizations, and resources
- `challenges/`: optional mini code-completion challenge and test

## Mini Enrichment
- Explain to a friend why `var` leaks out of blocks.
- Draw the scope in a notebook for an `if` block with `var`.
- Find one old code snippet online that still uses `var` and rewrite it with `let`/`const`.
