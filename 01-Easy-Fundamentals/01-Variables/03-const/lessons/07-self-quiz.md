# const Self Quiz

Score yourself: 1 point per correct answer.

## Questions

1. const is scoped to:

- A) Function
- B) Block
- C) Global only

2. True or false:

- const variables must be initialized when declared.

3. What happens here?

```js
const count = 1;
count = 2;
```

4. What happens here?

```js
const user = { name: "A" };
user.name = "B";
```

5. Which keyword should you default to for non-changing bindings?

- A) var
- B) let
- C) const

6. In one sentence, explain binding vs value.

## Answer Key

1. B
2. True
3. TypeError
4. Valid mutation
5. C
6. Binding is the variable reference; value is the data it points to.

## Interpretation

- 6/6: Excellent
- 4-5/6: Strong
- 0-3/6: Revisit core and visualization lessons
