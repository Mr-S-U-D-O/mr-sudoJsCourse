# 01-Variables

## Outcome
By the end of this module, you can choose between var, let, and const with confidence and explain why.

## Lessons
- 01-var: function scope, hoisting behavior, redeclaration risks.
- 02-let: block scope, temporal dead zone, safer reassignment.
- 03-const: immutable binding, mutable object values, intent signaling.

## Practice Loop
1. Read one lesson markdown file.
2. Implement the matching JavaScript function.
3. Run a focused test command.
4. Document one bug you prevented by choosing the right declaration.
