# Boss Fight - 01-Easy-Fundamentals

## Mission
Apply everything from Easy Fundamentals in one integrated assessment.

## Challenge Contract
Implement solveBossFight in boss.js.

Requirements:
- Input must be an array of submission objects.
- Each submission object contains id, score, and optional critical.
- A score of 70 or higher is a pass.
- Non-finite scores count as zero.
- Return topic, total, passedCount, failedCount, averageScore, and criticalFailures.

## Completion Standard
Run: npm run check 01-Easy-Fundamentals/99-Capstone-Boss-Fight
