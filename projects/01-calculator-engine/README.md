# Project 01: Calculator Engine

## What This Project Is

You are building a safe arithmetic expression engine that parses and evaluates expressions without using eval.

## Why This Project

This models real parser/evaluator work used in pricing rules, analytics formulas, and dynamic business logic systems.

## What You Should Know First

- JavaScript functions, arrays, loops, and objects
- Stack data structure basics
- Error handling fundamentals
- Unit testing concepts

## Rules

- Do not use eval or Function constructor.
- Keep parsing, validation, and execution separate.
- Throw clear errors for invalid syntax.
- Keep code deterministic and side-effect free.

## Motivation

If you can explain this parser under interview pressure, you demonstrate strong JavaScript fundamentals, algorithmic thinking, and code organization.
