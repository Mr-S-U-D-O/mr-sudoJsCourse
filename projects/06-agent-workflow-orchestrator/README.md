# Project 06: Agent Workflow Orchestrator

## What This Project Is

You are building an orchestrator that runs workflow graphs with dependencies, retries, and run-state tracking.

## Why This Project

This mirrors modern automation systems where multiple tool calls must be coordinated safely and reliably.

## What You Should Know First

- async/await and promises
- graph traversal basics
- retry/backoff patterns
- structured logging concepts

## Rules

- validate workflow schema before execution;
- keep step execution isolated;
- classify failures as retryable or terminal;
- persist state transitions for debuggability.

## Motivation

This project demonstrates advanced JavaScript architecture and automation engineering.
