# Project 03: Inventory Management System

## What This Project Is

You are building inventory domain logic for stock operations, reorder intelligence, and reporting.

## Why This Project

Inventory logic is core in ecommerce, warehouses, and supply chain systems. It demonstrates business-rule engineering.

## What You Should Know First

- JavaScript objects and classes/factories
- Data validation basics
- Immutable event logging concepts
- Aggregation/reporting patterns

## Rules

- Validate all incoming mutations.
- Keep an audit trail for stock changes.
- Separate data storage from domain logic.
- Fail loudly on invalid operations.

## Motivation

This project proves you can model business workflows, not just write algorithms.
